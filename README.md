## Nest Boilerplate

[Nest](https://github.com/nestjs/nest) starter project with user module, session module, jwt, firebase, mongo, rate limiting etc.

## Installation

```bash
$ yarn
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
